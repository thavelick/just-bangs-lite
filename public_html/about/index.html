About Just Bangs! Lite

Lightning-Fast Search Shortcuts

Just Bangs Lite is a client-side search tool that turns your browser into a powerful search launcher using bang shortcuts. Type stranger things jw! to find where to watch on JustWatch, or !e vintage vinyl records to search eBay for vintage vinyl records.

What Just Bangs Lite Does

Just Bangs Lite provides instant access to hundreds of websites through simple text shortcuts called "bangs". Instead of navigating to a website and then searching, you can search directly from one simple interface.

Client-Side Only

Runs entirely in your browser with zero server dependencies. No data ever leaves your device.

Dependency-Free

Pure HTML, CSS, and JavaScript. No frameworks, no build steps, no package.json.

Smart Dark Mode

Automatic dark mode detection with manual toggle. Respects your system preferences.

Mobile-First

Fully responsive design optimized for both desktop and mobile devices.

Configurable

Set your default search engine and customize your search experience.

Privacy-Focused

No tracking, no analytics, no data collection. Your searches stay private.

Example Usage

  • cats w! → Search Wikipedia for "cats"
  • !e vintage vinyl records → Search eBay for "vintage vinyl records"
  • stranger things jw! → Find where to watch on JustWatch
  • !g chocolate cake recipes → Search Google for "chocolate cake recipes"

Project Timeline

Original Just Bangs (2022-2025)
  • Early April 2022: Idiomdrottning conceived initial idea for a bang redirects website
  • April 7, 2022: Tristan coded up the original version in Python
  • April 8, 2022: Idiomdrottning added custom bang support (custom-bang.js)
  • April 24, 2022: Added dark mode support
  • September 2023: Added OpenSearch browser integration
  • 2025: Ongoing maintenance and refinements
Just Bangs Lite (July 2025)
  • July 18, 2025: Completed rewrite as client-side JavaScript application
  • July 18, 2025: Added modular architecture with dependency injection for testing
  • July 18, 2025: Implemented Jest unit testing without package.json
  • July 18, 2025: Added Biome for formatting/linting and GitHub Actions CI/CD
  • July 20, 2025: Added PWA features with service worker and environment-aware caching

  • July 20, 2025: Added mobile responsiveness and responsive design
